# Working Principles

## Language & UI
- The interface is in English.
- Dates are displayed in French format (FR locale).
- PDF contents are generated in French.

## Communication
- We communicate in French.

## Deployment
- You handle all deployments.
- Deployment steps are documented in `DEPLOY.md`.

## Compatibility
- The frontend must stay compatible with desktop and mobile (PWA). Changes should never break this requirement.
